Langkah 1: Upload & Preview File PDF

Klik untuk memilih file PDF dari komputer Anda

Maksimal 50MB

Memproses file...

Langkah 2: Hitung Biaya Cetak

Dari halaman: Sampai:
* 2 sisi hanya tersedia untuk Hitam Putih dengan Laser Jet

Ringkasan

Jumlah halaman efektif: 1
Total lembar: 1
Harga per lembar: Rp 500
Total: Rp 500

* Harga adalah perkiraan. Harga sebenarnya akan dikonfirmasi admin setelah cek file.

Jenis kertas: HVS 75 GSM

Langkah 3 : Konfirmasi via WhatsApp

Penting:
  • File akan kami cek manual oleh admin untuk memastikan jumlah halaman dan warna.
  • JANGAN BAYAR dulu sebelum mendapat konfirmasi harga final dari admin.
  • Admin akan membalas dalam 1-2 jam (jam kerja).

Klik tombol di bawah untuk mengirim detail order ke admin. Setelah chat terbuka, kirim file PDF yang sudah Anda preview melalui ikon attachment WhatsApp.

Setelah mengirim file, admin akan memproses order Anda.

Langkah 4 : Pembayaran QRIS

Tunggu Konfirmasi Admin!

Admin akan mengirimkan nominal harga final setelah cek file. Jangan transfer sebelum mendapat nominal dari admin.

Scan kode QR di bawah untuk melakukan pembayaran

SATU QRIS UNTUK SEMUA MedanPrint.Online

* Isi nominal sesuai instruksi admin saat transfer.

Langkah 5 : Cetak dan Ambil

Pesanan Anda sedang diproses. Status akan diinformasikan via WhatsApp.

Pesanan Selesai!

Silakan ambil di toko kami atau pesan ojek online.
