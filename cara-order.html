Langkah 1: Pilih File PDF

Klik untuk memilih file dari komputer Anda

Format PDF, maksimal 50MB (file akan dikirim via WhatsApp)

Langkah 2: Hitung Biaya Cetak

Ringkasan

Jumlah halaman efektif: 1
Total lembar: 1
Harga per lembar: Rp 300
Total: Rp 300

Langkah 3: Konfirmasi via WhatsApp

Klik tombol di bawah untuk mengirim detail order ke admin. Setelah chat terbuka, kirim file PDF yang sudah dipilih melalui attachment WhatsApp.

Setelah mengirim file dan konfirmasi, admin akan memproses order Anda.

Langkah 4: Pembayaran QRIS

Scan kode QR di bawah untuk melakukan pembayaran sebesar Rp 0

Atau transfer ke: Bank BCA 123456789 a.n. MedanPrint

Langkah 5: Cetak dan Ambil

Pesanan Anda sedang diproses. Status akan diinformasikan via WhatsApp.
