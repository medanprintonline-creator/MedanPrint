Cara Print PDF dari HP Android

Menggunakan Aplikasi Bawaan

  1. Buka file PDF yang ingin dicetak (misalnya dari Google Drive atau aplikasi File).
  2. Ketuk ikon tiga titik atau menu, lalu pilih "Cetak" atau "Print".
  3. Pilih printer yang tersedia (biasanya printer virtual seperti "Simpan sebagai PDF" atau printer Wi-Fi).
  4. Atur jumlah halaman, ukuran kertas, dan kualitas.
  5. Ketuk ikon cetak. Jika Anda menggunakan jasa online, pilih "Simpan sebagai PDF" untuk menghasilkan file baru, lalu kirim ke MedanPrint via WhatsApp.

Tips: Pastikan HP Anda terhubung ke printer yang sama jika ingin mencetak sendiri. Namun untuk hasil terbaik, gunakan jasa print profesional seperti MedanPrint.

Cara Print PDF dari iPhone (iOS)

  1. Buka file PDF di aplikasi File, Mail, atau Safari.
  2. Ketuk ikon bagikan (kotak dengan panah).
  3. Gulir ke bawah dan pilih "Cetak".
  4. Atur jumlah salinan dan rentang halaman.
  5. Lakukan gerakan "zoom out" pada pratinjau untuk mengakses opsi tambahan jika perlu.
  6. Ketuk "Cetak" di pojok kanan atas.
